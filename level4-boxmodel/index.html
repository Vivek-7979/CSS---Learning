Box model

  1. Content: The innermost part of the box where the actual content (text, images,video ,font etc.) is displayed.

  2. Padding: The space between the content and the border of the box.Sapce inside the itself element . (Apne box de andaro-andar hi spacing karni)

  3. Border: The line that surrounds the padding and content of the box.

  4. Margin: The space outside the border of the box, separating it from other elements.space form the other elements(Apne box de bahar spacing karni hor items toh) . Note :- The bigger value between the two margin items the bigger value will be applied.

NOTE :- 1.] Padding is defined in the clockwise direction (top, right, bottom, left) and can be set using the padding property in CSS.
2.] Border can be styled using properties like border-width, border-style, and border-color.
3.] Margin can also be defined in the clockwise direction (top, right, bottom, left) and can be set using the margin property in CSS.

Border

Border is the line that surrounds the padding and content of the box. It can be styled using properties like border-width, border-style, and border-color. The border can be solid, dashed, dotted, or double, among other styles. The width of the border can be specified in pixels, ems, or other units, and the color can be defined using color names, hex codes, or RGB values.

border= border-width border-style border-color
Example: border: 2px solid red;

Border box property [box sizing : border-box]

The border-box property is a CSS box model that includes the padding and border within the total width and height of an element. When using border-box, the width and height of an element are calculated to include the content, padding, and border, which can simplify layout calculations and prevent unexpected sizing issues. This is in contrast to the default content-box model, where the width and height only include the content, and padding and border are added on top of that.

border radius property is used to create rounded corners on elements. It can be applied to any element that has a border, and it allows you to specify the radius of the corners. The syntax for the border-radius property is as follows: border-radius: [top-left] [top-right] [bottom-right] [bottom-left];
