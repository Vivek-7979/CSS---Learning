To make responsive website we use RELATIVE UNITS

2.] EM ( Relative Unit )

  • em (generally dont have any specific full form ) is a relative unit that is based on the font size of the parent element.
  • 1em is equal to the current font size of the parent element.
  • If the font size of the element is 16px, then 1em is equal to 16px. If we set the font size of an element to 2em, it will be twice the size of the current font size. If we set it to 0.5em, it will be half the size of the current font size.

Second

3.] REM ( Relative Unit )

  • rem (root em) is a relative unit that is based on the font size of the root element (html).
  • 1rem is equal to the current font size of the root element.
  • If the font size of the root element is 16px, then 1rem is equal to 16px.
  • Example: If the root font size is 16px, then 1rem = 16px, 2rem = 32px, etc.
  • synatx : * body { font-size: 16px; means font size of whole body } . Now , #first{ font size : 2rem; means the first id named element has twice the size of root element i.e. 32px }

4.] VW/VH (view width/height)

  1. vw (view width) is a relative unit that is based on the width of the viewport.
  2. 1vw is equal to 1% of the viewport width. 50vw would be 50% of the viewport width.
  3. vh (view height) is a relative unit that is based on the height of the viewport.
  4. 1vh is equal to 1% of the viewport height. 50vh would be 50% of the viewport height.

POSITION PROPERTY

1.] Static Positioning (default)

  • Elements are positioned according to the normal flow of the document.
  • Top, right, bottom, left properties have no effect.

2.] Relative Positioning

  • Elements are positioned relative to their normal position.
  • Like : jithe apna element hona chaida c us toh 10px thora right nu krdo .

3.] Absolute Positioning

  • In absoulute positioning ; the element start from the left top corner of the page(or their nearest positioned ancestor) .

4.] Fixed Positioning

  • Elements are positioned relative to the viewport.
  • They remain in the same position even when the page is scrolled.
  • Like : apa jihna marji scroll karde rehiye oh chiz uthe hi fix rehni .
  • Example :- NAVIGATION BAR ( fix hi rehndi screen de top par ) ; HELP OPTION ( page de bottom pr fix rehndi )

4.] Z-index

  • z-index is a CSS property that controls the stacking order of elements on a web page.
  • Elements with a higher z-index value will be displayed in front of elements with a lower z-index value.
  • It only works on positioned elements (elements with a position value of relative, absolute, fixed, or sticky) bcz static elements cannot be stacked.
  • Example: If you have two overlapping elements, you can use z-index to control which one appears on top. An element with a z-index of 2 will appear in front of an element with a z-index of 1.
  • It have Integer value : -1, 0, 1, 2, ...
