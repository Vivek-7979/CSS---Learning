1.] Text align property

  • Text-align property is used to set the horizontal alignment of text in an element. It can take the following values:

    • left: Aligns the text to the left.
    • right: Aligns the text to the right.
    • center: Centers the text.
    • justify: Justifies the text, adding extra space between words as necessary to align both edges of the text with the left and right margins.

2.] Text decoration property

  • Text-decoration property is used to set the decoration of text in an element. It can take the following values:

    • none: Removes any text decoration.
    • underline: Adds a line under the text.
    • overline: Adds a line over the text.
    • line-through: Adds a line through the text.

3.] Text transform property

  1. none: Does not transform the text.
  2. uppercase: Converts the all text to uppercase.
  3. lowercase: Converts the all text to lowercase.
  4. capitalize: Capitalizes the first letter of each word.

4.] Line height property

  1. none: Does not set a specific line height.
  2. normal: Sets a default line height.
  3. number: Sets the line height to a multiple of the font size.
  4. length: Sets the line height to a specific length (e.g., 1.5em, 20px).
NOTE:- The line height = center of the first line to the center of the second line .

5.] Font property

A.] Font size : It is just used to control the size of the text or font.(where h1 to h6 tells us the importance of the text)

  1. 10px: Sets the font size to 10 pixels.
  2. 16px: Sets the font size to 16 pixels.
  3. 40px: Sets the font size to 40 pixels.

B.] Font weight : It is used to control the thickness or boldness of the text. { The value lies between 100 and 900 }

  1. normal: Sets the font weight to normal.
  2. bold: Sets the font weight to bold.[700]
  3. 100: Sets the font weight to a thin weight.
  4. 200: Sets the font weight to a light weight.
  5. 300: Sets the font weight to a book weight.
  6. 400: Sets the font weight to a regular weight.
  7. 500: Sets the font weight to a medium weight.
  8. 600: Sets the font weight to a semi-bold weight.
  9. 700: Sets the font weight to a bold weight.
  10. 800: Sets the font weight to an extra-bold weight.
  11. 900: Sets the font weight to a black weight.

C.]Font style property

  1. normal: Sets the font style to normal.
  2. italic: Sets the font style to italic.
  3. oblique: Sets the font style to oblique.

6.] Font Family

  1. Defines which font to use for the text. It can take a specific font name (e.g., Arial, Times New Roman) or a generic font family[jo ki hundi hai available] (e.g., serif, sans-serif, monospace).
  2. If the specified font is not available, the browser will use a fallback font from the same family(called- fallback mechanism).
  3. Web Safe Fonts : These are a set of fonts that are widely available across different operating systems and browsers. Examples include Arial, Times New Roman, Courier New, Verdana, and Georgia. Using web safe fonts ensures that your text will be displayed consistently for most users.

8.] Icons using Fonts ( by using : fontawesome.com or google fonts )

Icons can be added to a webpage using fonts, such as Font Awesome or Google Fonts. These font libraries provide a wide range of icons that can be easily integrated into your website. To use icons from these libraries, you typically need to include the appropriate CSS file in your HTML document and then use specific classes to display the desired icons.
