There are three types of CSS :-

  1. Inline CSS (applied inside the html tag using the style attribute) :- Generally we don't use this method as it makes the code less maintainable, less reusable and more cluttered.
  2. Internal CSS (apply inside the head tag between the "style" tag ):-More organised than inline , allows some reuse and hard to manage if the project grows large.
  3. External CSS ; means writing your styles in a separate file and linkking it to your HTML File. (most recommended ; In real life projects it is used mostly as it is resuable , maintainable and applied on multiple pages)

Types of CSS selectors

  1. Universal Selector [ * ](selects all elements on the page) ; saare hi page pr lag jana ||:- Example :- * {margin: 0; padding: 0;} (this will remove the default margin and padding from all elements)
  2. Element Selector[ "p" ] (selects all elements of a specific type) :- Example :- p {color: red;} (this will make the text colour of all the paragraphs red)
  3. ID Selector[ "#"] (selects the element with a specific ID attribute) It is very unnique . :- Example :- #idname {color: green;} (this will make the text colour of the element with the ID "idname" green)
  4. Class Selector[ "."] ; it allows grouping of multiple elements (selects all elements with a specific class name) :- Example :- .classname {color: blue;} (this will make the text colour of all the elements with the class "classname" blue)
  5. Grouping Selector[ ","] ; COMMA PAKE LIKHDO OH SAREYAN PR APPLY HO JANI CSS : it allows applying the same styles to multiple selectors at once. :- Example :- h1, h2, h3 {color: purple;} (this will make the text colour of all h1, h2, and h3 elements purple)
  6. Descendant Selector[ " " ] ; It selects elements that are descendants of a specified element. :- Example :- div p {color: pink;} (this will make the text colour of all paragraphs that are inside a div element pink)
NOTE :- ID selector has highest priority > class selector > Descendant selector > Grouping selector > Element selector(like selecting the whole element like - whole paragraph , whole h1 tag etc).
And universal selector has the least priority.
