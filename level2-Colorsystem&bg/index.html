2.] RGB : with the help of these 3 colours we can make any colour in the world .

  1. It consists of three values: red, green, and blue
  2. Utilized as rgb(r,g,b)
  3. Where r, g and b are values between 0 and 255 .
  4. like :
    1. rgb(255, 0, 0) for red
    2. rgb(0, 255, 0) for green
    3. rgb(0, 0, 255) for blue

3.] HEX : A hexadecimal color

  1. It is a 6-digit code(#RRGGBB) where that represents a color in the RGB color model.
  2. Each pair of digits (RedRed, GreenGreen, BlueBlue) represents the intensity of red, green, and blue components of the color, respectively.
  3. Here , 1 to 9 are 10 digits and A to F are 6 digits (total 16 digits) .
  4. It is used with using the # red red green green blue blue

5.] Absolute units

6.] Background Image

  1. Synatx of background image : background-image: url("image.jpg");
  2. Control image repeatation by using : background-repeat: repeat | no-repeat | repeat-x | repeat-y;
  3. Adjust the position of the background image using : background-position: top | bottom | left | right | center;
  4. Manipulate background-size: cover(scale the image to cover the entire background area, potentially cropping it) or background-size: contain (scale the image to fit within the background area without cropping, maintaining its aspect ratio).
  5. Background Attachment: Sets whether the background image scrolls with the content or remains fixed.
